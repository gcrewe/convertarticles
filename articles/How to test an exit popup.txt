How to test an exit popup
<h2>Issue</h2>
<p>It may be possible that you will be considering AB testing an <strong>Exit-Intent Popup</strong>. How do we <strong>AB Test </strong>one on <strong>Convert</strong>, or any similar widget or component that is implemented in javascript in our page? </p>
<h2>Solution</h2>
<p><strong>Convert </strong>is very good at adding javascript components, like exit popups into specific variations, therefore it is the ideal tool to add functionality to your experiment variations via javascript or HTML. In this article, we will show you how to this.</p>
<p>We decided to choose an open-source <strong><a href="https://github.com/beeker1121/exit-intent-popup" target="_self">Exit-Intent Popup</a> </strong>that is freely available for anyone use. <em>We do not endorse, guarantee, or recommend the linked software. We only show it for demonstration</em> purposes.<br><br>Firstly, we create a <strong>Convert AB Experiment</strong>. Then, while in the <strong>Visual Editor</strong>, we open our <strong>Variation</strong> <strong>Custom JS </strong>editor:</p>
<p> <img src="https://support.convert.com/hc/article_attachments/360060289391/blobid0.png" alt="blobid0.png"></p>
<p>Then copy the minified <strong>Exit Intent Pop Up </strong>minified javascript code in the editor. After that add <a href="https://github.com/beeker1121/exit-intent-popup#template" target="_blank" rel="noopener">the template code</a>. It should look like the following image:</p>
<p><img src="https://support.convert.com/hc/article_attachments/360060291771/blobid2.png" alt="blobid2.png"><br><br>Then apply the changes and save it. <br><br>Then have a preview of your variation with the <strong>Live Preview</strong> within the <strong>Variation Menu:<br><br><img src="https://support.convert.com/hc/article_attachments/360060290511/blobid1.png" alt="blobid1.png"></strong><br><br>Your page where you created will show. Try to navigate out of it, and your popup will show:<br><br><img src="https://support.convert.com/hc/article_attachments/360060118812/blobid3.png" alt="blobid3.png"><br><br>You can customize the code to your desire. </p>
<p>Then you can <strong>Save It </strong>and <strong>Activate it.</strong></p>
<p><strong><img src="https://support.convert.com/hc/article_attachments/360060118992/blobid4.png" alt="blobid4.png"></strong><strong><br></strong>You will see your report showing both variations<strong> Running</strong>.</p>
<p>Now you should be able to see if your <strong>Exit-Intent Popup</strong> leads into more conversions.<strong> </strong></p>
