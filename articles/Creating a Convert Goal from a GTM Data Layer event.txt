Creating a Convert Goal from a GTM Data Layer event
<h2>Issue Explanation</h2>
<p>Sometimes we may have already tracked conversions through GTM using dataLayer events, and we would like to use these conversion events to create conversion goals in Convert.</p>
<h2>Instructions</h2>
<p>We can do this by implementing a <em>GTM tag</em> with a Convert <em>JavaScript Triggered Goal</em> code that would be used to trigger the GTM dataLayer event that we want to track.</p>
<p>First, let say that an event called <em>TestAction </em>is being pushed into the dataLayer in the following way:</p>
<p><em>window.dataLayer = window.dataLayer || []</em><br><em>window.dataLayer.push({ </em><br><em> event: 'TestAction'</em><br><em>});</em></p>
<ol>
<li>
<strong>Create the Convert Javascript Triggered Goal:</strong><br><br>Open your experiment on the Experience Summary and click on the Goal Editor to open it. Access the Goal Templates:<br><br><img src="https://support.convert.com/hc/article_attachments/5990662894093/mceclip0.png" alt="mceclip0.png"><br><br>Select the JavaScript Triggered Goal Template:<br><br><img src="https://support.convert.com/hc/article_attachments/5990902819725/mceclip1.png" alt="mceclip1.png"><br><br>Name your new goal and save it. Open your newly-created goal. Select and copy the JavaScript code from the goal and paste it somewhere to later create your GTM Tag.<br><br><img src="https://support.convert.com/hc/article_attachments/6449327349005/mceclip0.png" alt="mceclip0.png"><br><br>
</li>
<li>Go to your Google Tag Manager account at <a href="https://tagmanager.google.com/">https://tagmanager.google.com/</a> and select the account you are going to work on.<br><br>Click on the "ADD A NEW TAG" button:<br><br><br><img src="https://support.convert.com/hc/article_attachments/360016100232/blobid0.png" alt="save image" width="608" height="294"><br><br>Click on the "Choose a tag type to begin setup":<br><br><img src="https://support.convert.com/hc/article_attachments/360016100652/blobid8.png" alt="save image" width="611" height="297"><br><br>Scroll down and select the "Custom HTML" option.<br><br><img src="https://support.convert.com/hc/article_attachments/360016100732/blobid9.png" alt="save image" width="610" height="295"><br><br>Paste the code from the Convert JavaScript Triggered Goal into the HTML Section and name your tag:<br><br><img src="https://support.convert.com/hc/article_attachments/360016201531/blobid2.png" alt="save image" width="651" height="407"><br><br>When the following dialog opens, select the "ADD TRIGGER" option:<br><br><img src="https://support.convert.com/hc/article_attachments/360016100992/blobid12.png" alt="save image" width="297" height="132"><br><br>A list of possible triggers should appear.  Click on the "+" icon on the upper left, and select the trigger type:<br><br><img src="https://support.convert.com/hc/article_attachments/360016101452/blobid15.png" alt="save image" width="603" height="293"><br><br>Scroll down and select the "Custom Event" option:<br><br><img src="https://support.convert.com/hc/article_attachments/360016201871/blobid4.png" alt="save image" width="605" height="314"><br><br>Re-save your whole Tag, and you are done.</li>
<li>Test your goal by triggering the GTM dataLayer event, and see if the goal is triggered in your experiment. You can also access Goals by clicking on Goals in the top tab next to Experiences.<br><img src="https://support.convert.com/hc/article_attachments/4423902043533/mceclip3.png" alt="mceclip3.png">
</li>
</ol>
